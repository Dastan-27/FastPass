# FastPass Uni

FastPass Uni es una solución integral de *Pre-Order* diseñada para eliminar los cuellos de botella en la cafetería universitaria. La plataforma permite a la comunidad académica gestionar sus pedidos de forma remota, optimizando el tiempo de descanso y mejorando la eficiencia operativa de los puntos de venta.

---

## Funcionalidades Principales

### 1. Visualización de Disponibilidad en Tiempo Real
* **Propósito:** Eliminar la frustración del usuario ante la falta de stock.
* **Historia de Usuario:** "Como estudiante, quiero ver el menú actualizado y el stock disponible desde mi celular para no perder tiempo eligiendo opciones agotadas."
* **Requerimiento Funcional:** Integración con el inventario para actualizar el estado del "Menú del Día" en tiempo real.

### 2. Pedido Anticipado (Pre-Order)
* **Propósito:** Descentralizar la toma de pedidos.
* **Historia de Usuario:** "Como estudiante con poco tiempo, quiero realizar mi pedido desde el salón para que esté listo al llegar."
* **Requerimiento Funcional:** Selección de productos de múltiples puntos de venta y generación de orden vinculada al usuario.

### 3. Pago Digital Integrado
* **Propósito:** Suprimir las filas en las cajas físicas.
* **Historia de Usuario:** "Como usuario, quiero pagar digitalmente para dirigirme directamente a la zona de entrega."
* **Requerimiento Funcional:** Pasarela de pagos segura que valide la transacción antes de enviar la orden a cocina.

### 4. Sistema de Notificaciones "Ready to Pick"
* **Propósito:** Evitar aglomeraciones en la zona de entrega.
* **Historia de Usuario:** "Como estudiante, quiero recibir una alerta cuando mi pedido esté listo para no esperar de pie."
* **Requerimiento Funcional:** Notificaciones push automáticas cuando el estado cambie a "Listo para entrega".
